PATS is a pyAutoTest project skeleton, which will be used as a template when create a new python functional test project.

How To

1. copy PATS to some where at your local hard drive.
 
2. rename PATS to a new name for you testing project.

3. under your testing project folder, rename libraries/PATS folder to a new name, where is the location to store your test project specific libraries files.

4. run setup.py under the folder.

5. add functional test cases under tests folder. 

6. modify TestSuite.py to include test cases.

7. run TestSuite.py to execute all selected tests.

8. examine the test result in tests_report folder.

9. you done!
