PATS is a pyAutoTest project skeleton, which will be used as a template when create a new python functional test project.

How To

1. copy PATS to some where at your local hard drive.
 
2. rename PATS to a new name for you testing project.

3. run setup.py under the folder.

4. add functional test cases under tests folder. 

5. modify TestSuite.py to include test cases.

6. run TestSuite.py to execute all selected tests.

7. examine the test result in tests_report folder.

8. you done!
